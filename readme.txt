	1. Install nodeJS from https://nodejs.org/
	2. execute "npm search" command in terminal inside your folder
	3. execute "npm install bower" command in your teminal inside your folder
	4. Create a "server.js" to run the node server
	5. Create the "settings.json"
	6. Create the "deploy.js" for deployment script
	7. Create "test.js" for all your test scripts
	8. Create "build.js" for all the build process
